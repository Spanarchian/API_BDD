## Features
- Encrypted user authorizaton
- Database initialization
- New user signup
- User login/logout
- User settings
- Modern user interface
- Bulma framework
- Limited custom css/js
- Easily customizable

## Setup
``` 
git clone https://github.com/KarhooColin/API_BDD.git
cd API_BDD
pip install -r requirements.txt
python app.py
```
